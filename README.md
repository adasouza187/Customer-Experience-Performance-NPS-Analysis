# 📊 Customer Experience Analytics – NPS Evolution & Performance

## 📌 Visão Geral
Este projeto apresenta um dashboard desenvolvido em Power BI para análise da evolução do NPS (Net Promoter Score), com foco na comparação de desempenho em relação ao ano anterior 

O relatório foi construído com o objetivo de apoiar a tomada de decisão estratégica, permitindo identificar tendências, variações e oportunidades de melhoria na experiência do cliente.

---

## 🎯 Objetivos do Projeto
- Monitorar a evolução do NPS ao longo do tempo  
- Comparar o desempenho atual com períodos anteriores 
- Identificar padrões e variações por diferentes dimensões de negócio  
- Apoiar decisões orientadas a dados  

---

## 📊 Principais Análises
- Evolução mensal do NPS  
- Comparativo anual 
- Análise por gerente  
- Segmentação por tipo de cliente  
- Performance por região  

---

## 🧠 Principais Insights
- Identificação de tendências de satisfação do cliente ao longo do tempo  
- Comparação de performance entre diferentes períodos  
- Identificação de áreas com potencial risco ou oportunidade  
- Apoio na priorização de ações estratégicas  

---

## 🛠️ Tecnologias Utilizadas
- Power BI  
- DAX (Data Analysis Expressions)  
- Power Query (ETL)  
- HTML aplicado em visuais personalizados  

---

## 🧩 Modelagem de Dados
O projeto foi estruturado utilizando boas práticas de modelagem dimensional (modelo estrela), garantindo:

- Melhor performance  
- Facilidade de manutenção  
- Escalabilidade das análises
